# Adminidor

A [Next.js](https://nextjs.org) application bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app), using the App Router, TypeScript, and Tailwind CSS.

## Getting Started

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result. Edit `src/app/page.tsx` to start building — the page auto-updates as you save.

## Scripts

| Command         | Description                                  |
| --------------- | -------------------------------------------- |
| `npm run dev`   | Start the dev server (Turbopack) on port 3000 |
| `npm run build` | Create a production build                     |
| `npm run start` | Serve the production build                    |
| `npm run lint`  | Run ESLint                                    |
